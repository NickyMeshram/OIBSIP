# Retail Sales Data Analysis

## About the Project

In this project, I performed Exploratory Data Analysis (EDA) on a retail sales dataset to understand customer behaviour, sales trends, and product performance. The analysis was done using Python and different data visualization techniques to identify useful business insights.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## What I Did

- Loaded and inspected the dataset
- Checked for missing values and data types
- Calculated descriptive statistics
- Analyzed monthly and quarterly sales trends
- Studied customer demographics such as age groups and gender
- Compared revenue across different product categories
- Created a correlation heatmap
- Added an extra visualization to understand sales by gender
- Summarized the findings and suggested business recommendations

## Key Findings

- Sales changed across different months and quarters, showing seasonal trends.
- Some product categories generated more revenue than others.
- Most customers belonged to specific age groups, which can help in targeted marketing.
- Customer demographics and sales patterns can be used to make better business decisions.

## Dataset

Retail Sales Dataset

> **Note:** This dataset contains **Product Category** instead of individual **Product Name**. Therefore, the product analysis has been carried out at the category level.
